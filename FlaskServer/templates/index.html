Bienvenue sur la démo Flask de l'API IA Multi-Origines

Cette démonstration présente un système d'IA multi-origines adaptatif qui peut utiliser différentes sources d'intelligence artificielle en fonction de la disponibilité et des besoins :

  • OpenAI - Utilise l'API OpenAI avec GPT-4o
  • Local - Utilise une implémentation locale sans appel d'API externe
  • Auto - Sélectionne automatiquement la meilleure option disponible

Vous pouvez changer la source d'IA en utilisant le menu déroulant en haut à droite.

    Documentation API

    Référence de l'API
    Configuration de l'origine IA

    Endpoint pour obtenir ou définir l'origine IA par défaut.

    GET /api/ai/config/origin
    // Réponse
{
  "success": true,
  "data": {
    "origin": "auto",
    "name": "Auto",
    "description": "Sélectionne automatiquement la meilleure option disponible"
  }
}
    POST /api/ai/config/origin
    // Requête
{
  "origin": "openai"
}

// Réponse
{
  "success": true,
  "data": {
    "origin": "openai",
    "name": "OpenAI",
    "description": "Utilise l'API OpenAI (GPT-4o)"
  },
  "message": "Default AI origin set to OpenAI"
}
    Analyse de sentiment
    POST /api/ai/sentiment
    // Requête
{
  "text": "Je suis très content de ce produit !",
  "origin": "auto"  // Optionnel, par défaut utilise l'origine définie globalement
}

// Réponse
{
  "success": true,
  "data": {
    "rating": 5,  // Note de 1 à 5
    "confidence": 0.85,  // Confiance entre 0 et 1
    "origin": "openai"  // L'origine utilisée pour l'analyse
  }
}
    Génération de résumé
    POST /api/ai/summary
    // Requête
{
  "text": "Long texte à résumer...",
  "origin": "auto"  // Optionnel
}

// Réponse
{
  "success": true,
  "data": {
    "summary": "Résumé du texte...",
    "origin": "openai"
  }
}
    Recommandations de produits
    POST /api/ai/recommendations
    // Requête
{
  "description": "Je cherche un nouveau smartphone avec un bon appareil photo",
  "origin": "auto"  // Optionnel
}

// Réponse
{
  "success": true,
  "data": {
    "recommendations": [
      "iPhone 15 Pro",
      "Samsung Galaxy S23",
      "Google Pixel 7",
      "Xiaomi Redmi Note 12"
    ],
    "origin": "openai"
  }
}
